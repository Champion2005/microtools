# microtools

Bunch of vibe coded microtools for random tasks.

## Stack

- Vite
- React
- Tailwind CSS

## Getting Started

1. Install dependencies:
	- `npm install`
2. Start local dev server:
	- `npm run dev`
3. Build for production:
	- `npm run build`

## MCP Server

The homepage links to the companion MCP server here: [Champion2005/microtools-mcp](https://github.com/Champion2005/microtools-mcp)
