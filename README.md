# microtools

Bunch of vibe coded microtools for random tasks.

## Stack

- Vite
- React
- Tailwind CSS

## Getting Started

1. Install dependencies:
	- `npm install`
2. Start local dev server:
	- `npm run dev`
3. Build for production:
	- `npm run build`

## Copilot Customization

This repo includes workspace Copilot assets for web development under `.github/`:

- `copilot-instructions.md`
