# RegEx Playground

## Purpose
A real-time tester with syntax highlighting and match extraction to help developers dry-run patterns.

## Features
- Real-time pattern matching against multiline text.
- Match group extraction.
- Searchable cheat sheet for common patterns (email, URL, dates, etc.).

## Implementation Details
- Standard Javascript `RegExp` object for matching.
- Match results mapped into visual blocks.
- Pre-defined templates for quick loading.
